Detailed Description

template<typename DistanceUnit>
class rmb::SparkMaxVelocityController< DistanceUnit >

A wrapper around the SparkMax motorcontroller that allows for the user to set and get the velocity of the motor accurately through PID functionallity

◆ ConversionUnit

template<typename DistanceUnit >
using rmb::SparkMaxVelocityController< DistanceUnit >::ConversionUnit = typename units::compound_unit<DistanceUnit, units::inverse<units::radians> >

The ratio between the user-defined DistanceUnit and radians. This is used for converting to and from user-defined units and raw units. For example, if a wheel has a wheel with a circumference of 2 meters, the conversion from meters to radians (2pi = 1 rotation) would be: 

2 meters / 2pi radians
(1/pi) m/rad
conversion ≈ 0.3183

◆ RawUnit

template<typename DistanceUnit >
using rmb::SparkMaxVelocityController< DistanceUnit >::RawUnit = typename units::unit<std::ratio<2>, units::radians, std::ratio<1> >

The SparkMax does it's distances in rotations. The rotations quantity is based on radians. The math for the conversion is as follows: 

1 rotation * 2pi rad = 2pi rad

Thus the multiplier provided to the units:: library is 2pi
